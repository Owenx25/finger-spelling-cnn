In order to run this project Jupyter must be installed along with the following python libraries:
  - numpy
  - pandas
  - matplotlib
  - seaborn
  - torchvision
  - torch
  - cuda
 After that the sign_mnist_train.csv and sign_mnist_test.csv must be in the same directory as the notebook.
 
 After these steps the notebook can be run with Jupyter to run individual cells.
